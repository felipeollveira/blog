# SISTEMA DE BLOG

### O que o usuario pode fazer
 - Acessar as resoluções de problemas
- Ler posts
    - Comentar posts
 - Enviar e-mails

### O que o usuario não pode fazer
 - Fazer publicações
 -  Comentar em outros comentarios (respostas)
  -  Curtir
